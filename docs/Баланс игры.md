## 📊 Система прогрессии персонажа

### Ранги и характеристики

| Ранг | Диапазон сущности | Стартовая сущность | Макс. артефакты | Макс. способности | Условия прорыва |
|------|-------------------|-------------------|------------------|-------------------|-----------------|
| 1 | 100-300 | 100 | 1 | 1 | Доступен с начала |
| 2 | 300-600 | 300 | 1 | 2 | 1d100 > 50 + ивент/зелье |
| 3 | 600-900 | 600 | 2 | 3 | 1d100 > 60 + ивент/зелье |
| 4 | 900-1200 | 900 | 2 | 4 | 1d100 > 70 + ивент/зелье |
| 5 | 1200-1500 | 1200 | 3 | 5 | 1d100 > 80 + ивент/зелье |
| Чемпион 1 | 1500-2000 | 1500 | 3 | 6 | 1d100 > 85 + легендарный ивент |
| Чемпион 2 | 2000-2500 | 2000 | 3 | 6 | 1d100 > 90 + легендарный ивент |
| Чемпион 3 | 2500-3000 | 2500 | 4 | 6 | 1d100 > 50 + божественный ивент |

### Стартовые характеристики нового персонажа
- **Сущность**: 100 (основное здоровье)
- **Защита**: 100 (первый слой здоровья)
- **Энергия**: 100 (для действий и тренировок)
- **Скорость**: обычная (15 метров за ход)
- **Инвентарь**: 10 слотов
- **Деньги**: 100 монет
- **Бонусы**: все равны 0

## ⚔️ Механика боя

### Основные формулы расчёта

#### 1. Попадание (атака)
```
Бросок попадания = 1d(сущность_персонажа + сущность_оружия + бонусы_точности - штраф_атаки)
Минимальный результат = минимальный_бросок_ранга (обычно 1)
```

#### 2. Уклонение (защита)
```
Бросок уклонения = 1d(сущность_цели + бонусы_уклонения - штраф_брони - штраф_уклонения)
Минимальный результат = минимальный_бросок_ранга (обычно 1)
```

#### 3. Определение попадания
- **Попадание**: если бросок попадания > бросок уклонения
- **Промах**: если бросок попадания ≤ бросок уклонения
- **Ничья**: побеждает защищающийся

#### 4. Расчёт урона
```
Базовый урон = сущность_оружия (или 5 для безоружия)
Финальный урон = макс(0, базовый_урон - игнорирование_брони - сопротивление_урону)
```

### Система пробития брони

| Тип пробития | Уровень | Пробиваемая броня |
|--------------|----------|-------------------|
| NONE | 0 | Никакую |
| LIGHT | 1 | Лёгкую |
| MEDIUM | 2 | Лёгкую, среднюю |
| HEAVY | 3 | Лёгкую, среднюю, тяжёлую |
| VERY_HEAVY | 4 | Любую, включая сверхтяжёлую |

### Характеристики брони

| Тип брони | Штраф уклонения | Штраф попадания | Игнорирует урона | Штраф скорости |
|-----------|-----------------|-----------------|-------------------|----------------|
| Лёгкая | 0 | 0 | 50 | 0 |
| Средняя | 100 | 50 | 200 | 0 |
| Тяжёлая | 250 | 150 | 500 | 3м/ход |
| Сверхтяжёлая | 400 | 300 | 1000 | 6м/ход |

### Скорость передвижения

| Тип скорости | Дистанция за ход | Бонусы/штрафы |
|--------------|------------------|----------------|
| Очень медленная | 0-5м | -20 к уклонению |
| Медленная | 0-10м | -10 к уклонению |
| Обычная | 0-15м | нет |
| Быстрая | 0-20м | +10 к уклонению |
| Очень быстрая | 0-30м | +20 к уклонению |

## 🏋️ Система прокачки

### Тренировка персонажа
- **Кулдаун**: 24 часа
- **Прирост сущности**: 1-20 (равномерное распределение)
- **Стоимость**: 20 энергии
- **Ограничение**: до максимальной сущности ранга
- **Минимальный бросок**: 1d20 + 1

### Прокачка оружия
- **Прирост сущности**: 1-5 за тренировку
- **Кулдаун**: 24 часа
- **Максимум**: зависит от редкости оружия
- **Боевой опыт**: автоматическое повышение при использовании в бою

## 🎯 Правила создания монстров

### Основная формула баланса
```
Базовая сущность монстра = (Макс_сущность_ранга × 1.5) + (Ранг × 50)
```

### Принципы баланса
1. **Пропорциональность**: сила монстра растёт плавно с ростом ранга
2. **Компенсация**: скорость компенсирует разницу в сущности
3. **Предсказуемость**: игрок понимает чего ожидать от монстра ранга X
4. **Разнообразие**: разные типы поведения и тактик

### Шаблон создания монстра
```json
{
  "id": "mon-[тип]-[подтип]",
  "name": "Название монстра",
  "rankOrder": 1-8,
  "baseEssence": число,
  "speedId": "speed-[тип]",
  "description": "Описание монстра",
  "skills": ["skill-id-1", "skill-id-2"],
  "lootTable": [
    {
      "templateId": "item-template-id",
      "chance": 0.0-1.0,
      "minQuantity": 1,
      "maxQuantity": 1
    }
  ]
}
```

## 📈 Рекомендуемые диапазоны сил

### 🟢 Ранг 1 (100-300 сущности игрока)
- **Лёгкие монстры**: 80-150 сущности
  - Примеры: гоблины, крысы, мелкие звери
  - Скорость: быстрая/очень быстрая
  - Лут: базовые материалы
  
- **Средние монстры**: 120-200 сущности
  - Примеры: волки, бандиты, скелеты
  - Скорость: обычная/быстрая
  - Лут: простое оружие, материалы
  
- **Тяжёлые монстры**: 150-250 сущности
  - Примеры: кабаны, огры, зомби
  - Скорость: медленная/обычная
  - Лут: редкие материалы, броня

### 🟡 Ранг 2 (300-600 сущности игрока)
- **Лёгкие монстры**: 250-400 сущности
  - Примеры: эльфы-разбойники, тени, демоны-слабые
  - Скорость: очень быстрая
  - Лут: магические компоненты
  
- **Средние монстры**: 350-500 сущности
  - Примеры: орки, гоблины-воины, элементали
  - Скорость: обычная
  - Лут: редкое оружие, зелья
  
- **Тяжёлые монстры**: 450-650 сущности
  - Примеры: тролли, големы, демоны-средние
  - Скорость: медленная
  - Лут: эпические материалы, артефакты

### 🟠 Ранг 3 (600-900 сущности игрока)
- **Лёгкие монстры**: 500-700 сущности
  - Примеры: вампиры-юные, призраки, ассасины
  - Скорость: очень быстрая
  - Лут: магические свитки, редкие ингредиенты
  
- **Средние монстры**: 650-850 сущности
  - Примеры: некроманты, демоны-сильные, элементали-старшие
  - Скорость: обычная/быстрая
  - Лут: эпическое оружие, артефакты
  
- **Тяжёлые монстры**: 800-1100 сущности
  - Примеры: личи, демоны-вожди, драконы-молодые
  - Скорость: медленная/обычная
  - Лут: легендарные материалы, уникальные предметы

### 🔴 Ранг 4 (900-1200 сущности игрока)
- **Лёгкие монстры**: 800-1000 сущности
  - Примеры: тени-убийцы, вампиры-древние, демоны-элиты
  - Скорость: очень быстрая
  - Способности: невидимость, телепортация
  
- **Средние монстры**: 1000-1300 сущности
  - Примеры: личи-магистры, демоны-князья, драконы-взрослые
  - Скорость: обычная
  - Способности: мощные заклинания, ауры
  
- **Тяжёлые монстры**: 1200-1600 сущности
  - Примеры: рыцари смерти, древние големы, драконы-старшие
  - Скорость: медленная
  - Способности: регенерация, массовые атаки

### ⚫ Чемпионы (1200+ сущности игрока)
- **Чемпион 1**: 1400-1800 сущности

## ⚖️ Компенсационные механики

### Скорость vs Сущность
| Скорость | Модификатор сущности | Описание |
|----------|---------------------|----------|
| Очень медленная | +30% | Больше HP, но легко попасть |
| Медленная | +15% | Больше HP, но легче попасть |
| Обычная | 0% | Стандартный баланс |
| Быстрая | -20% | Меньше HP, но сложнее попасть |
| Очень быстрая | -30% | Меньше HP, но очень сложно попасть |

### Специальные способности и модификаторы

| Способность | Модификатор сущности | Описание |
|-------------|---------------------|----------|
| Регенерация | +15% | Восстанавливает HP каждый ход |
| Яд/Кровотечение | -10% | Наносит периодический урон |
| Щит/Броня | +20% | Дополнительный слой защиты |
| AoE атаки | +25% | Атакует по площади |
| Невидимость | -15% | Сложно обнаружить и атаковать |
| Телепортация | -10% | Может менять позицию |
| Оглушение | +20% | Может пропустить ход противника |
| Призыв | -25% | Призывает помощников |

### Групповые монстры
При создании групповых монстров:
- **Уменьшение сущности**: -30% для каждого монстра в группе
- **Координация**: групповые атаки и тактики
- **Лут**: увеличенное количество за счёт количества

## 📊 Динамика сложности

### Принципы плавной прогрессии

#### 1. Линейный рост с экспоненциальными элементами
- **Базовый рост**: ~30-50% между рангами
- **Экспоненциальные скачки**: на рубежах чемпиона
- **Сглаживание**: дополнительные монстры между основными рангами

#### 2. Предсказуемость с вариативностью
- **Средняя сложность**: предсказуема для ранга
- **Вариативность**: ±20% от среднего значения
- **Экстремальные случаи**: редкие монстры с нестандартными характеристиками

#### 3. Компенсаторный баланс
- **Слабость в одном** компенсируется **силой в другом**
- **Быстрые но хрупкие** vs **медленные но прочные**
- **Магические** vs **физические** типы урона

### Рекомендуемая кривая сложности

```
Ранг 1: 100-300 (разброс 200)
Ранг 2: 250-500 (разброс 250)
Ранг 3: 450-750 (разброс 300)
Ранг 4: 700-1100 (разброс 400)
Ранг 5: 1000-1500 (разброс 500)
Чемпион 1: 1300-1800 (разброс 500)
Чемпион 2: 1700-2300 (разброс 600)
Чемпион 3: 2200-3000 (разброс 800)
```

### Текущие проблемы и решения

#### Проблема 1: Резкий скачок Orc
- **Текущий**: 120 → 450 (+275%)
- **Рекомендуемый**: добавить монстров 200-300 сущности
- **Решение**: создать "Разбойник-ветеран" (250 сущности), "Орк-воин" (320 сущности)

#### Проблема 2: Недостаток разнообразия
- **Текущий**: 1-2 монстра на ранг
- **Рекомендуемый**: 3-5 монстров на ранг
- **Решение**: добавить монстров с разными тактиками

#### Проблема 3: Отсутствие способностей
- **Текущий**: все монстры без способностей
- **Рекомендуемый**: 1-3 способности на монстра
- **Решение**: реализовать систему способностей монстров

#### Проблема 4: Несбалансированный лут
- **Текущий**: лут не соответствует сложности
- **Рекомендуемый**: лут пропорционален сложности
- **Решение**: пересмотреть таблицы лута

## 🔧 Практические рекомендации

### Для разработчиков
1. **Используйте формулы** для расчёта базовой сущности
2. **Добавляйте компенсации** за скорость и способности
3. **Тестируйте монстров** против персонажей целевого ранга
4. **Создавайте группы** монстров с разными ролями

### Для гейм-дизайнеров
1. **Думайте о тактике**: как игрок будет сражаться с монстром?
2. **Добавляйте разнообразие**: не все монстры должны быть похожими
3. **Учитывайте окружение**: локация может влиять на сложность
4. **Балансируйте лут**: награда должна соответствовать риску

### Для балансировки
1. **Тестируйте на разных рангах**: монстр должен быть сложен но проходим
2. **Собирайте статистику**: как часто игроки умирают от монстра
3. **Корректируйте постепенно**: не делайте резких изменений
4. **Прислушивайтесь к игрокам**: их фидбэк важен
