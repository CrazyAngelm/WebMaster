## 🎯 Основные принципы экономики

### Ключевые концепции
- **Экспоненциальный рост цен** - каждая редкость значительно дороже предыдущей
- **Балансировка по сущности** - базовая стоимость зависит от мощи предмета
- **Сложность и материалы** - крафт учитывает стоимость компонентов и работы
- **Прогрессивная доступность** - предметы соответствуют рангу игрока

### Базовая стоимость
- **1 единица сущности = 5 монет**
- **1 единица прочности = 50 монет**

## 📐 Математические формулы

### Общая формула ценообразования
```
Цена = (БазоваяСтоимость × Параметр × МножительРедкости × МножительТипа) + СтоимостьМатериалов
```

### Формулы по типам предметов

#### Оружие
```
ЦенаОружия = (5 × Сущность × МножительРедкости × МножительТипаОружия) + СтоимостьМатериалов
```

#### Броня
```
ЦенаБрони = (50 × Прочность × МножительРедкости × МножительТипаБрони) + СтоимостьМатериалов
```

#### Расходники (зелья, свитки)
```
ЦенаРасходника = (3 × Сущность × МножительРедкости) + СтоимостьМатериалов
```

#### Сумки
```
ЦенаСумки = (100 × КоличествоСлотов × МножительРедкости) + СтоимостьМатериалов
```

#### Материалы
```
ЦенаМатериала = БазоваяЦена × МножительРедкостиМатериала
```

## 🧪 Зелья прорыва - Уникальная система компонентов

### Принцип ценообразования
```
ЦенаЗельяПрорыва = (СтоимостьКомпонентов × 1.5) + (Сложность × 1000 × МножительРедкости)
```

**Важно:** Зелья прорыва **не рассчитываются по сущности**, так как их ценность определяется **уникальными компонентами**, которые добываются только с монстров определённого ранга.

4. **Стимулирование** - разные виды деятельности сопоставимы по доходности
5. **Масштабируемость** - система легко расширяется новыми предметами

## 📈 Ожидаемые результаты

- **COMMON предметы**: доступны для всех, 1-10 часов игры
- **RARE предметы**: достижимы для новичков, 10-50 часов игры
- **EPIC предметы**: значимая цель, 50-200 часов игры
- **MYTHIC предметы**: высшая цель для опытных, 200-1000 часов
- **LEGENDARY предметы**: месяцы игры, статусный предмет
- **DIVINE предметы**: легендарный статус, годы игры или уникальные события

Эта экономическая система обеспечивает стабильную и интересную игровую экономику, где цена любого предмета соответствует его реальной ценности и усилиям, необходимым для его получения.
