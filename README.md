# WebMaster

Игровой проект на React + TypeScript + Vite.

## Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Проект будет доступен по адресу: **http://localhost:5173**

### Дополнительные команды

- **Сборка проекта:**
```bash
npm run build
```

- **Предпросмотр собранного проекта:**
```bash
npm run preview
```

## Технические детали

- **Порт:** 5173
- **Сборщик:** Vite
- **Фреймворк:** React 18
- **Язык:** TypeScript
- **Стили:** Tailwind CSS

## Требования

- Node.js 18+ (рекомендуется)
